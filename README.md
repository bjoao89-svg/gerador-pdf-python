
# Gerador de PDF em Python

Este projeto é um aplicativo simples desenvolvido em Python que permite gerar arquivos PDF a partir de um texto digitado pelo usuário.

## Funcionalidades
- Interface gráfica simples
- Geração de arquivos PDF
- Texto personalizado pelo usuário

## Tecnologias utilizadas
- Python
- Tkinter
- ReportLab

## Observação
Este repositório contém apenas o código-fonte do projeto.
